# 📱 Calculadora Mobile

Aplicativo de calculadora desenvolvido em **React Native**, com foco em aplicar conceitos de lógica, estados e interação de interface em dispositivos móveis.

---

## 🚀 Funcionalidades

* Inserção de números
* Operações básicas:

  * Soma (+)
  * Subtração (-)
  * Multiplicação (×)
  * Divisão (÷)
* Botão de igual (=)
* Botão AC (limpar tudo)
* Interface simples e intuitiva

---

## 🧠 Lógica da Calculadora

A aplicação funciona baseada em três estados principais:

| Estado   | Função                |
| -------- | --------------------- |
| `num`    | Número atual digitado |
| `result` | Resultado armazenado  |
| `opera`  | Operação selecionada  |

Fluxo de funcionamento:

1. Usuário digita o primeiro número
2. Escolhe uma operação
3. Digita o segundo número
4. Pressiona "=" para calcular
